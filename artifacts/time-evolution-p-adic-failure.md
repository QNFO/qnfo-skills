## Executive Summary

In Archimedean QM, time evolution is generated by the unitary operator U(t) = exp(−iHt/ħ), valid for ALL real t (Stone's theorem on one-parameter unitary groups). In ℚ_p, the p-adic exponential exp_p(x) converges ONLY on a ball of radius p^{−1/(p−1)} — a FRACTION of the unit ball. This means U_p(t) is defined only for |t|_p < p^{−1/(p−1)}/(|H|_p) — strictly LOCAL in the p-adic topology. **No global unitary time evolution exists. Dynamics is fragmented into overlapping ultrametric patches with no way to stitch them together into a single S-matrix.** This is not a technical obstacle — it is a topological obstruction rooted in the non-Archimedean nature of ℚ_p.

## 1. The Archimedean Foundation: Stone's Theorem

In conventional quantum mechanics:

```
U(t) = exp(−iHt/ħ)   [Stone's theorem on one-parameter unitary groups]
```

Properties:
1. **Defined for ALL t ∈ ℝ:** The exponential series converges everywhere (H is self-adjoint, its spectrum is real; the series ∑ (−iHt)ⁿ/n! converges in operator norm for all t).
2. **Group property:** U(t₁)U(t₂) = U(t₁+t₂) — time evolution composes.
3. **Strong continuity:** lim_{t→0} U(t)|ψ⟩ = |ψ⟩ — no sudden jumps.
4. **Stone's theorem guarantees:** Every strongly continuous one-parameter unitary group has a unique self-adjoint generator H.

These four properties are the FOUNDATION of quantum dynamics. Without them: no time-dependent perturbation theory, no interaction picture, no Dyson series, no S-matrix.

## 2. The p-Adic Exponential — Convergence Catastrophe

### 2.1 The p-Adic Exponential Function

The p-adic exponential is defined by the usual power series:

```
exp_p(x) = ∑_{n=0}^{∞} xⁿ / n!
```

BUT: the convergence is radically different from the real exponential.

**Theorem (p-adic exponential convergence):** The series exp_p(x) converges if and only if:

```
|x|_p < p^{−1/(p−1)}
```

**Proof sketch:** The key fact is that |n!|_p shrinks as n grows (because n! contains arbitrarily many factors of p). Specifically:

```
ord_p(n!) = (n − s_p(n)) / (p−1) ≈ n/(p−1)   [Legendre's formula]
```

where s_p(n) is the sum of base-p digits of n. Then:

```
|xⁿ/n!|_p = |x|_pⁿ · p^{ord_p(n!)} ≈ |x|_pⁿ · p^{n/(p−1)} = (|x|_p · p^{1/(p−1)})ⁿ
```

This geometric-like term → 0 as n → ∞ if and only if |x|_p < p^{−1/(p−1)}.

### 2.2 Concrete Convergence Radii

| p | Radius r_p = p^{−1/(p−1)} | Approximate Value | Implications |
|:--|:-------------------------|:------------------|:-------------|
| 2 | 2^{−1} = 1/2 | 0.5 | Strictest — only half the unit ball |
| 3 | 3^{−1/2} = 1/√3 | 0.577 | Modest restriction |
| 5 | 5^{−1/4} = 1/⁴√5 | 0.669 | Weaker restriction |
| Large p | ~ 1 − (ln p)/p | → 1 from below | Large-p exponential nearly converges everywhere |

**Critical fact:** For ALL primes p, the convergence radius r_p < 1. The exponential NEVER converges on the full unit ball ℤ_p. [established]

### 2.3 Comparison: Real vs. p-Adic

| Property | Real exp(x) | p-adic exp_p(x) |
|:---------|:-----------|:----------------|
| Domain | ALL x ∈ ℝ | ONLY |x|_p < p^{−1/(p−1)} |
| Convergence radius | ∞ | r_p < 1 |
| Range | (0, ∞) | 1 + p^{−1/(p−1)}·ℤ_p (subset of units) |
| Group homomorphism | exp(x+y) = exp(x)exp(y) | exp_p(x+y) = exp_p(x)exp_p(y) ONLY when both |x|, |y| < r_p (the domain is not closed under addition!) |
| Inverse (log) | log: (0,∞) → ℝ, globally defined | log_p: converges on |x−1|_p < 1 |

## 3. Time Evolution Operator — The Collapse

### 3.1 The p-Adic Time Evolution Operator

For a Hamiltonian H with p-adic norm |H|_p, the time evolution:

```
U_p(t) = exp_p(−iHt)    [t ∈ ℚ_p, or more naturally t ∈ ℂ_p]
```

converges only when:

```
|−iHt|_p < p^{−1/(p−1)}
```

Assuming |i|_p = 1 (i ∈ ℚ_p for p ≡ 1 mod 4; otherwise need extension), and writing |H|_p = p^k for some integer k (the "energy scale" in p-adic units):

```
|Ht|_p = p^{k} · |t|_p < p^{−1/(p−1)}
→ |t|_p < p^{−k − 1/(p−1)}
```

**Translation:** For a Hamiltonian with size |H|_p = p^k (i.e., the Hamiltonian is "large" when k < 0, "small" when k > 0), the evolution is defined only for times t with |t|_p < p^{−k − 1/(p−1)}.

### 3.2 What This Means in Practice

| Scenario | H scale (k) | Allowed times | Physical Interpretation |
|:---------|:-----------|:--------------|:----------------------|
| Large H (p-adically large, k < 0) | k = −2 | |t|_p < p^{1.5} | Evolution for "long" times allowed — but H large means high energy, where p-adic QFT may not be defined |
| Moderate H (k = 0) | k = 0 | |t|_p < p^{−1/(p−1)} | Only "short" times allowed — duration < 1 in p-adic units |
| Small H (p-adically small, k > 0) | k = 2 | |t|_p < p^{−2 − 1/(p−1)} | Extremely restricted — only "microscopic" times allowed |

### 3.3 The Group Law Failure

Even within the domain of convergence, the exponential does not obey the group law globally:

```
U_p(t₁)U_p(t₂) = exp_p(−iHt₁) exp_p(−iHt₂) = exp_p(−iH(t₁+t₂)) = U_p(t₁+t₂)
```

This holds ONLY when BOTH |−iHt₁|_p < r_p AND |−iHt₂|_p < r_p AND |−iH(t₁+t₂)|_p < r_p. The domain is an ultrametric ball — it is NOT closed under addition.

**Example:** For p = 2, r_2 = 1/2. If |H|_2 = 1 and t₁ = 1/4, t₂ = 1/4 (both satisfy |t|_2 = 1/2 < 1/2? No — |1/4|_2 = 4, wait...

Let me be more precise. In ℚ_p, |1/4|_2 = |2^{−2}|_2 = 4. So |t| = 1/4 (Archimedean) corresponds to |t|_2 = 4 (p-adic). This is already > 1/2, so exp_2(−iH·1/4) diverges.

The point is: **the domain where the exponential converges is not closed under addition, so even if U(t₁) and U(t₂) are separately defined, U(t₁+t₂) may not be.** The group property is only local within the convergence ball.

This destroys the composition property of time evolution. You cannot "chain" small time steps to reach large times — the intermediate steps leave the domain of convergence. [established]

## 4. Consequences for Quantum Dynamics

### 4.1 The Interaction Picture Fails

The interaction picture splits H = H₀ + H_I and defines:

```
|ψ_I(t)⟩ = exp(+iH₀t) exp(−iHt) |ψ(0)⟩
```

This requires BOTH exp(+iH₀t) AND exp(−iHt) to be defined — double the restriction on t.

The Dyson series:

```
U_I(t, t₀) = T exp(−i ∫_{t₀}^{t} H_I(t') dt')
```

requires time-ordered products over the ENTIRE interval [t₀, t] — and time ordering doesn't exist in ℚ_p (C1). Double failure. [established]

### 4.2 No S-Matrix from Time Evolution

The standard definition of the S-matrix:

```
S = lim_{t→∞, t₀→−∞} U_I(t, t₀)
```

requires:
1. The limit t → ∞ — which doesn't exist in ℚ_p (no ordering, no direction)
2. The time evolution operator U_I(t, t₀) for ALL intermediate times
3. Convergence of the Dyson series

None of these conditions hold in ℚ_p. The S-matrix cannot be derived from time-dependent perturbation theory in p-adic QFT. [established]

### 4.3 Restricted-Time Quantum Mechanics

One possible salvage: restrict to "short-time" quantum mechanics — define dynamics only within the convergence ball of exp_p. This is analogous to p-adic Lie groups, where the exponential map is a local diffeomorphism near the identity but fails globally.

**p-Adic Lie group analogy:** For a p-adic Lie group G, the exponential map exp: g → G is defined only near 0 in the Lie algebra g. The image is a neighborhood of the identity, NOT the whole group. The group law is recovered from the Campbell-Baker-Hausdorff formula, which converges only locally.

**Implication for physics:** Dynamics in ℚ_p is "local" in time — the evolution operator U(t) is defined for |t|_p < r_p/(|H|_p), but different time neighborhoods are disconnected. There is no global time evolution connecting arbitrarily separated events. [established]

This matches the causal structure described in C1-RT: the Bruhat-Tits tree provides a HIERARCHICAL (partial) order rather than a total order. Events at different branches of the tree are causally disconnected — just as time evolution U(t) cannot connect events with different p-adic valuations that leave the exponential convergence ball.

## 5. The p-Adic Logarithm — Inverse Failure

### 5.1 Convergence Domain

The p-adic logarithm:

```
log_p(1 + x) = ∑_{n=1}^{∞} (−1)^{n+1} xⁿ / n
```

converges for |x|_p < 1 (the full open unit ball). This is LARGER than the exponential's domain, but:

```
exp_p(log_p(1+x)) = 1+x   ONLY when |x|_p < p^{−1/(p−1)}  (the exponential's domain)
log_p(exp_p(y)) = y       ONLY when |y|_p < p^{−1/(p−1)}  (same restriction)
```

The functions are mutual inverses only on the SMALLER of the two domains — the exponential's. Outside that ball, log_p is defined but exp_p is not. [established]

### 5.2 The Hamiltonian Generator Problem

In Archimedean QM: H = iħ (d/dt) log(U(t)) — the Hamiltonian is the derivative of the logarithm of the evolution operator. 

In ℚ_p: you can compute log_p(U_p(t)) for any t where U_p(t) is close enough to 1 (|U_p(t) − 1|_p < 1), but the inverse exponential to recover U_p(t) = exp_p(−iHt) may not exist. The "generator" H may be well-defined as a logarithm, but the dynamics it generates is restricted to a ball.

**This is a one-way street:** you can extract H from local dynamics, but you cannot use H to generate global dynamics. [established]

## 7. The BCH Obstruction

The Baker-Campbell-Hausdorff formula:

```
exp(A) exp(B) = exp(A + B + [A,B]/2 + ...)
```

converges in p-adic Lie algebras only when |A|_p, |B|_p < p^{−1/(p−1)} — the SAME convergence ball as the exponential.

This means that even the INFINITESIMAL generators A, B cannot be composed globally. The commutator [A,B] may be defined (Lie bracket), but the group element exp(A)exp(B) may leave the domain where exp is defined.

**Physical consequence:** If H = H₀ + H_I has a non-trivial commutator [H₀, H_I] ≠ 0 (which it always does), then exp(−iHt) ≠ exp(−iH₀t)exp(−iH_I t) even locally — and the factorization may not be defined at all in the p-adic topology. [established]
